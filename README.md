### Usage
1. install

```bash
# use npm
npm install flexible-layout --save

# use yarn
yarn add flexible-layout

# use script
<script src="flexible-layout/index.js" />
```

2. require / import

```bash
# use esm
import 'flexible-layout';

# use cmd
require('flexible-layout');
```
